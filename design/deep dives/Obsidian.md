Obsidian is a free, local-first note-taking app using Markdown files for building interconnected knowledge bases.




**Core Features:**
- Links: Bidirectional [[wiki-style]] connections between notes.
- Graph View: Visualizes note relationships for pattern discovery.
- Canvas: Infinite board for brainstorming and diagramming.
- Tags (#) and folders for organization.
- Attachments: Embed images, PDFs, audio; supports embeds like iframes.

**Note-Taking:**
- Markdown editor with previews; daily notes via plugins.
- Quick switcher and command palette for navigation.
- Web clipper for capturing sites as Markdown.

**Document Management:**
- Vaults as local folders; open .md format for portability.
- Search, sort by date/tags; version history with Sync add-on.
- Bases plugin for database-like tables.

**Plugins & Customization:**
- 2,000+ community plugins (e.g., Kanban, Dataview for queries, Calendar, Tasks).
- 160+ themes; extensible for workflows like Zettelkasten.

**Syncing & Pricing:**
- Free core app; Sync ($4-5/mo) for encrypted cross-device.
- Publish ($8-10/mo) for online sharing.
- Manual sync via cloud storage.

**Pros:**
- Secure, offline; no lock-in; highly customizable.
- Powerful for personal knowledge management.

**Cons:**
- Steep learning curve; no native collaboration; no web app.

**Use Cases:**
- Personal wiki; research; writing planning; ADHD-friendly organization via maps/plugins.
