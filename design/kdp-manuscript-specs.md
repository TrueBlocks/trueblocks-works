## 1. Accepted File Formats

| Format | Recommended For | Notes                                  |
| ------ | --------------- | -------------------------------------- |
| PDF    | Print books     | Preferred; must be print-ready         |
| DOCX   | Print books     | Converted by KDP; less control         |
| DOC    | Print books     | Legacy format; DOCX preferred          |
| EPUB   | eBooks          | For Kindle; KDP converts automatically |
| KPF    | eBooks          | Kindle Create output; best for eBooks  |

**For our workflow:** Export as DOCX, then convert to PDF locally for print submission.

## 2. Print Options

Before uploading your manuscript and cover, you'll choose ink/paper type, trim size, bleed setting, and cover finish.

### 2.1 Ink and Paper Type

#### Paperback Options

| Ink Type       | Paper Type  | Paper Weight          |
| -------------- | ----------- | --------------------- |
| Black ink      | Cream paper | 50-61 lb (74-90 GSM)  |
| Black ink      | White paper | 50-61 lb (74-90 GSM)  |
| Premium color  | White paper | 60-71 lb (88-105 GSM) |
| Standard color | White paper | 50-61 lb (74-90 GSM)  |

#### Hardcover Options

| Ink Type      | Paper Type  | Paper Weight          |
| ------------- | ----------- | --------------------- |
| Black ink     | Cream paper | 50-61 lb (74-90 GSM)  |
| Black ink     | White paper | 50-61 lb (74-90 GSM)  |
| Premium color | White paper | 60-71 lb (88-105 GSM) |

**Notes:**

- Paper weight varies by printing location
- Black ink books can use cream or white paper
- Color ink books only available on white paper (higher image contrast)
- Standard color is NOT available for hardcover
- Color printing costs more than black ink; paper type doesn't affect cost

**Our default:** Black ink on white paper (essays are text-only).

### 2.2 Cover Finish

Covers are printed on 80 lb (220 GSM) white paper stock:

| Finish | Appearance                   | Typical Use                       |
| ------ | ---------------------------- | --------------------------------- |
| Glossy | Shiny; darker blacks         | Textbooks, children's, nonfiction |
| Matte  | Minimal sheen; polished look | Novels, fiction, literary works   |

**Note:** Hardcover books are printed as case laminate.

**Our default:** Matte finish (literary essays).

### 2.3 Printing Costs

- **Regular trim size:** Up to 6.12" (155.5mm) wide × 9" (228.6mm) tall
- **Large trim size:** Exceeds either dimension (higher printing cost)
- 6" × 9" is a regular trim size (lower cost)

For detailed costs, see:

- [Paperback Printing Cost](https://kdp.amazon.com/help?topicId=G201834340)
- [Hardcover Printing Cost](https://kdp.amazon.com/help?topicId=GHT976ZKSKUXBB6H)

## 3. Trim Sizes (US)

KDP supports these standard trim sizes for paperbacks:

| Size (inches) | Size (mm)     | Common Use                     |
| ------------- | ------------- | ------------------------------ |
| 5" × 8"       | 127 × 203     | Mass market paperback          |
| 5.25" × 8"    | 133 × 203     | Digest                         |
| 5.5" × 8.5"   | 140 × 216     | Trade paperback (small)        |
| **6" × 9"**   | **152 × 229** | **Trade paperback (standard)** |
| 6.14" × 9.21" | 156 × 234     | Royal                          |
| 6.69" × 9.61" | 170 × 244     | Crown Quarto                   |
| 7" × 10"      | 178 × 254     | Textbook                       |
| 7.5" × 9.25"  | 190 × 235     | Executive                      |
| 8" × 10"      | 203 × 254     | Workbook                       |
| 8.5" × 11"    | 216 × 279     | Letter (US standard)           |

**Our default:** 6" × 9" (152.4 × 228.6 mm) — ideal for essays and poetry collections.

**Custom sizes (paperback only):** Width 4"-8.5" (10.16-21.59 cm), Height 6"-11.69" (15.24-29.69 cm).

## 4. Margin Requirements

Margins depend on page count (affects gutter/inside margin due to binding).

### 3.1 Minimum Outside Margins

| Edge                 | Minimum (inches) | Minimum (mm) |
| -------------------- | ---------------- | ------------ |
| Top                  | 0.25"            | 6.35         |
| Bottom               | 0.25"            | 6.35         |
| Outside (left/right) | 0.25"            | 6.35         |

### 3.2 Inside (Gutter) Margin by Page Count

| Page Count | Min Inside Margin (inches) | Min Inside Margin (mm) |
| ---------- | -------------------------- | ---------------------- |
| 24–150     | 0.375"                     | 9.53                   |
| 151–300    | 0.5"                       | 12.7                   |
| 301–500    | 0.625"                     | 15.88                  |
| 501–700    | 0.75"                      | 19.05                  |
| 701–828    | 0.875"                     | 22.23                  |

### 3.3 Recommended Margins for 6" × 9" (300-400 pages)

| Edge    | Recommended | Notes                          |
| ------- | ----------- | ------------------------------ |
| Top     | 0.5"        | Room for running header        |
| Bottom  | 0.5"        | Room for page number           |
| Outside | 0.5"        | Comfortable reading margin     |
| Inside  | 0.75"       | Gutter for binding (350 pages) |

## 5. Page Count Limits

| Type           | Minimum | Maximum |
| -------------- | ------- | ------- |
| Black & white  | 24      | 828     |
| Premium color  | 24      | 550     |
| Standard color | 24      | 550     |

**Note:** Page count must be divisible by 2 (front and back of each sheet). KDP adds blank pages if needed.

## 6. Bleed Requirements

### 5.1 No Bleed (Standard)

- Content stays within margins
- White border around all pages
- **Use for:** Text-only books, essays, poetry

### 5.2 With Bleed

For images or color extending to page edge:

| Edge    | Bleed Amount    |
| ------- | --------------- |
| Top     | 0.125" (3.17mm) |
| Bottom  | 0.125" (3.17mm) |
| Outside | 0.125" (3.17mm) |
| Inside  | None (gutter)   |

**Our workflow:** No bleed for manuscript interiors (cover has bleed).

## 7. Font Requirements

### 6.1 Embedding

- **All fonts must be embedded** in the PDF
- Subset embedding is acceptable
- No font substitution allowed

### 6.2 Recommended Fonts

| Category | Font Families                                |
| -------- | -------------------------------------------- |
| Serif    | Garamond, Times New Roman, Georgia, Palatino |
| Sans     | Arial, Helvetica, Open Sans                  |
| Mono     | Courier, Consolas (for code samples)         |

### 6.3 Minimum Sizes

| Element   | Minimum Size |
| --------- | ------------ |
| Body text | 9pt          |
| Footnotes | 7pt          |
| Headers   | 8pt          |

**Recommended body text:** 11-12pt for comfortable reading.

## 8. Image Requirements

### 7.1 Resolution

| Use Case       | Minimum DPI | Recommended DPI |
| -------------- | ----------- | --------------- |
| Line art       | 300         | 600             |
| Photos (B&W)   | 300         | 300             |
| Photos (color) | 300         | 300             |

### 7.2 Color Space

| Interior Type | Color Space                  |
| ------------- | ---------------------------- |
| Black & white | Grayscale                    |
| Color         | CMYK preferred, RGB accepted |

### 7.3 Sizing

- Images should not exceed the printable area (inside margins)
- No scaling over 100% — always use full resolution source

## 9. PDF Settings

When exporting to PDF for KDP:

### 8.1 Required Settings

| Setting     | Value                       |
| ----------- | --------------------------- |
| PDF Version | 1.3 – 1.7 (1.4 recommended) |
| Fonts       | Embedded (all)              |
| Resolution  | 300 DPI minimum             |
| Color Space | CMYK or RGB                 |
| Compression | Lossless for images         |

### 8.2 Forbidden Content

- ❌ Crop marks
- ❌ Registration marks
- ❌ Color bars
- ❌ Printer's marks
- ❌ Annotations
- ❌ Form fields
- ❌ JavaScript
- ❌ Transparency (flatten before export)

### 8.3 PDF/X-1a Compliance

KDP accepts PDF/X-1a:2001 compliant files (recommended for professional results).

## 10. Content Structure

### 9.1 Required Elements

| Element      | Required | Notes               |
| ------------ | -------- | ------------------- |
| Title page   | Yes      | First page (recto)  |
| Copyright    | Yes      | Verso of title page |
| Body content | Yes      | The actual book     |

### 9.2 Optional Front Matter (in order)

1. Half-title page
2. Series page / Also By
3. Title page
4. Copyright page
5. Dedication
6. Epigraph
7. Table of Contents
8. Preface / Foreword
9. Introduction

### 9.3 Optional Back Matter (in order)

1. Acknowledgements
2. Notes / Endnotes
3. Bibliography
4. Glossary
5. Index
6. About the Author
7. Also By / Other Books

### 9.4 Page Number Guidelines

| Section      | Page Numbers                        |
| ------------ | ----------------------------------- |
| Front matter | Roman numerals (i, ii, iii) or none |
| Body         | Arabic numerals (1, 2, 3)           |
| Back matter  | Continue Arabic from body           |

## 11. Common Rejection Reasons

| Issue                      | Solution                              |
| -------------------------- | ------------------------------------- |
| Fonts not embedded         | Embed all fonts before PDF export     |
| Images below 300 DPI       | Replace with higher resolution source |
| Content outside margins    | Adjust layout; check all pages        |
| Crop/printer marks present | Disable marks in PDF export settings  |
| Wrong page size            | Set document size to exact trim size  |
| Password-protected PDF     | Remove all PDF security               |
| Corrupted file             | Re-export from source document        |
| File too large             | Compress images; limit is 650 MB      |

## 12. File Size Limits

| Type     | Maximum Size |
| -------- | ------------ |
| Interior | 650 MB       |
| Cover    | 40 MB        |

---

## 13. Pre-Flight Checklist

Before uploading to KDP:

- [ ] Page size matches selected trim size exactly
- [ ] All fonts embedded (use PDF preflight tool)
- [ ] Images are 300+ DPI
- [ ] No content in bleed area (unless bleed enabled)
- [ ] All content within safe margins
- [ ] No crop/registration marks
- [ ] PDF is not password-protected
- [ ] Page count is even number
- [ ] Front matter uses Roman numerals or no numbers
- [ ] Body text starts on page 1 (Arabic)
- [ ] File size under 650 MB
